# 🧠 Spring AI App

A Spring Boot application integrating AI capabilities using all AI providers. This project demonstrates how to build
AI-powered REST endpoints using the **Spring AI** library.

> 📺 Featured in our YouTube
> video: [Watch Here](https://youtube.com/playlist?list=PLagGeTxGWF3EWCm4HhBL-NDaaoAeu6BPT&si=aP0dW2cB-YbL572h)  
> ⭐ Don't forget to star the repo if you find it useful!

---

## 🚀 Features

- REST API to interact with AI models
- Spring Boot + Spring AI integration
- Easy-to-extend structure for future enhancements
- Environment-based configuration for API keys

---

## 🛠️ Tech Stack

- Java 17+
- Spring Boot
- Spring AI
- Maven
- OpenAI / Azure / HuggingFace / Ollama (configurable)

---

## ⚙️ Setup Instructions

### 1. Clone the repo

```bash
git clone https://github.com/VengatX-YT/spring-ai-app
cd spring-ai-app
```
